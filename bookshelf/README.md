A full-stack web application built with **Django** that allows users to browse, search, and read books directly in the browser. It features a dark-mode UI inspired by Spotify.

## Live Demo
**https://bluntkey.pythonanywhere.com**

## Features
* **User Authentication:** Secure Login/Logout and Registration system.
* **In-App Reader:** Read PDFs directly within the browser (no downloads needed).
* **Smart Search:** Search books by **Title** OR **Author**.
* **Personalization:** Users can select books to save them to their "My Library" collection.

## Tech Stack
* **Backend:** Python, Django
* **Frontend:** HTML5, CSS3, JavaScript
* **Database:** SQLite (Local/Dev)
* **Media Handling:** Pillow (Images), PDF handling
* **Deployment:** PythonAnywhere
